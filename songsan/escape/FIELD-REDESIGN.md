# 믿음의 항해 — 현장·선생님 상호작용 재설계서

작성: 프로젝트 총괄 기획 / 검토: 설계안 4종(렌즈1~4)·심사 3건 종합
대상: 대표 보고용 · songsan/escape

---

## 1. 한 장 요약

- **기존**: 5개 자물쇠 전부 "어딘가서 숫자카드 찾기 → 패드에 4자리 입력". 교사 미니게임 3종도 결국 "이기면 숫자도장 → 패드 입력". **감각도 행동도 6막 내내 동일.**
- **재설계**: 6막이 **여섯 개의 서로 다른 감각·사회 채널**을 쓴다 — 입(제창) → 손(매듭) → 귀(리듬) → 몸(사슬-해방) → 눈(필름 합체) → 팀간 계승(조립·파송). **공유 4자리 숫자 자물쇠 0개.**
- **패드는 얇아진다**: 진짜 신규 UI는 **2막 리듬탭 딱 하나**. 나머지는 확인버튼·기존 코드필드(문자 허용)·개입없음. 5막은 클수록 패드가 사라져 "숫자로 시작 → 목소리·불로 끝"난다.

## 2. 막별 상호작용 표

### 2-1. 메커니즘이 전부 다르다 (한눈에)

| 막 | 감각·역학 채널 | 핵심 동사 | 패드 |
|---|---|---|---|
| 0막 부두 | **입 (목소리)** | 함께 소리내어 고백한다 | 확인버튼 |
| 1막 선장실 | **손 (손기술)** | 매듭을 묶고 함께 돛을 편다 | 개입없음(또는 확인) |
| 2막 조타실 | **귀 → 손 (청각·타이밍)** | 빛 신호를 듣고 리듬으로 친다 | **리듬탭(신규)** |
| 3막 화물칸·무덤섬 | **온몸 (재연)** | 서로의 사슬을 끊어준다 | 개입없음 |
| 4막 선원실 | **눈 (시각 합체)** | 4장을 겹쳐 한 얼굴을 드러낸다 | 코드필드(단어) |
| 5막 등대 | **협업 + 팀간 계승** | 부품을 끼워 불을 켜고 파송된다 | 개입없음(선포로 끝) |

> 6막 어디에도 "숫자 찾아 4자리 입력"이 반복되지 않는다. 사회 역학은 5막(팀간 계승) 한 곳에만, 조립은 5막 한 곳에만 몰아 가족 겹침을 차단했다.

## 3. 막별 상세 (시간순 흐름 · 4명의 분산과 합류)

**0막 — 승선 서약.** 도착 즉시 4명이 부두 이젤 앞에 선다. 서약문 4행을 미리 나눠 맡아 한 명씩 또렷이 이어 읽고, 마지막 행은 넷이 손 모아 제창한다. 제창 직후 각자 명부에 이름과 고백 한 줄을 **동시에** 자필 서명한다(한 명이 캐리 못 하게). 선장이 손등에 승선 스탬프를 찍으면 패드에서 확인 1탭 → 선장실이 열린다. *첫 행위가 목소리라 시작부터 태블릿 독점이 원천 차단된다.*

**1막 — 출항 삭구.** 선장실 삭구 밧줄에 닻·돛·계류·신호 4지점이 라벨돼 있다. 넷이 각자 자기 지점으로 흩어져 도해 카드대로 매듭 1개씩 동시에 묶고, 다 묶이면 양끝을 나눠 잡고 갑판장 구령에 도르래로 돛천을 **부드럽게** 함께 올린다(줄다리기 아님). 갑판장이 매듭 4개를 실물 검사하고 통과 시 **나무 십자가**를 손에 쥐여 준다 — 이 십자가가 3막 상륙의 열쇠다.

**2막 — 폭풍 침로.** 폭풍음이 패드에서 몰아친다. 전령 1명만 조명 낮춘 복도 끝 신호수에게 가 손전등 모스를 받아적어 돌아온다(정보 비대칭 → 팀 대화 강제). 서기는 눅8·막4를 대조해 침로 방향값을 확정하고, 정찰은 배의 종으로 리듬을 되쳐 검증한다. 항해사가 그 장단을 **패드 리듬탭에 그대로 두드리면** 폭풍음이 뚝 끊기고 "잠잠하라" 자막이 뜬다.

**3막 — 무덤섬 상륙.** 4명이 종이 리본으로 서로 얽힌 채 시작한다(=쇠사슬에 매인 거라사인). **자기 사슬은 자기가 못 끊는다.** 한 명씩 "나를 묶는 것"을 한마디(또는 침묵) 고백하면 동료가 그 리본을 끊어준다. 전원 풀리면 1막의 십자가를 해안에 세우고 함께 선포한다. 거라사 교사가 확인하고 admin으로 상륙을 처리하며 **실물 자물쇠 열쇠 + 기름·부싯돌(5막 부품)**을 지급한다. *잡아당기기 금지 — 리본은 손으로만 끊긴다.*

**4막 — 그 한 사람.** 각자 반투명 필름 1장(군중 그림의 1/4)을 들고 흩어져 조건을 읽는다. 서기가 "열두 해"(혈루증)와 "열두 살"(야이로 딸) 조건을 소리내 좁히는 동안, 넷이 다시 모여 **네 모서리를 동시에 맞잡아** 창/손전등 빛에 비춘다. 정렬이 맞아야 군중 속 한 얼굴이 붉게 떠오르고 "그 한 사람"이 읽힌다 → 코드필드에 단어 입력. 자기검증형이라 교사 판정이 필요 없다.

**5막 — 등대 점화·파송.** 패드가 암전되고 "불은 너희 손에 있다"만 남는다. 문 앞에서 눅9 짐검사(휴대폰·가방을 바구니에)를 마친 넷이 등대 사방에 선다. 항해 내내 모은 부품 — 카트리지·십자가 횃대·기름·부싯돌 — 을 각자 순서대로 딸깍 끼우고, 전원 손을 얹어 "예수를 바라보자!"를 **선포**하면 교사가 LED를 점등한다. 켠 팀은 즉시 다음 팀의 등대지기로 파송된다.

## 4. 선생님 재설계 — "이기면 숫자" 틀 탈출

**핵심 수술**: 교사 미니게임의 결과가 더는 "숫자 도장"이 아니다. 교사는 채점기가 아니라 **장면의 인물이자 심판**이 되고, 통과는 admin에서 처리하거나 실물을 손에 쥐여 주는 것으로 끝난다.

| 기존 | 재설계 |
|---|---|
| 매듭 시험 이기면 **숫자 도장** → 패드 입력 | 1막: 갑판장이 실물 매듭을 도해와 **육안 검사** → 십자가(실물) 지급. 숫자 세탁 제거 |
| 모스 신호수 → 숫자 알아내 입력 | 2막: 빛 모스를 듣고 **리듬으로 재현**(타이밍 입력). 신호수는 정답 발화 금지 |
| 가위바위보 사슬 이기면 숫자 | 3막: **서로의 사슬을 끊어주는** 고백-해방 재연 → admin 상륙 처리 + 실물 열쇠 지급 |

**교사 6명 배치·동선**:
- 0막 **선장**(부두) · 1막 **갑판장**(선장실) · 2막 **신호수**(복도끝, 조명↓) · 3막 **거라사**(무덤섬 부스) · 4막 **유동 선원**(자기검증형이라 상주 불필요, 3·5막 지원) · 5막 **폐회+등대지기**(등대)
- **인력 절감 장치**: ① 4막이 자기검증형 → 교사 1명을 아껴 병목인 3·5막에 투입. ② 5막 파송-계승으로 **먼저 클리어한 팀이 다음 팀의 스태프**가 되어, 1차 웨이브만 6인 풀가동 후 인력이 배로 늘어난다.
- **병목 관리**: 3막이 팀당 시간이 길어 최대 병목 → 하중상자 서브과제를 **삭제**하고 사슬-해방+십자가에 초점을 압축, 가장 노련한 교사를 배치.

## 5. 패드에 새로 필요한 것

| 항목 | 유형 | 난이도 | 비고 |
|---|---|---|---|
| **리듬탭 위젯** (2막) | (c) 신규 UI | **중** | 타이밍 캡처 + 허용오차 판정. 유일한 진짜 신규 코드. 기존 모스(82425) 인프라 재활용 |
| 확인버튼 변형 "했다/받았다" (0막) | (b) 신규(경) | **하** | physOpen에 `confirm:true` 분기 ~15줄 |
| 코드필드 문자 허용 (4막 "그 한 사람") | (a) 기존 재활용 | **하** | 숫자→문자 허용만 추가, 신규 UI 아님 |
| admin "이 팀 통과" 버튼 정리 (3·5막) | 기존 재활용 | **하** | 기존 phys/item/advance 토글 정리 |

**패드 없이 되는 것**: 1막(매듭·완전 오프라인), 3막(교사 admin+실물 지급), 5막 점화(패드 밖, 선포로 끝). → **3개 막이 오프라인**이라 태블릿 독점이 구조적으로 불가능.

**규율**: 신규 UI는 2막 리듬탭 **하나로 못박고** 다른 막에 새 UI를 만들지 않는다.

*보조 레버*: 리듬탭 신규 코드가 부담되면, 2막을 기존 `type:'dial'`을 8방위 나침반으로 **리스킨(신규 아님)**하는 방위 다이얼로 대체 가능. 단 감각 참신도(타이밍 vs 회전)는 리듬탭이 우위 — 기본은 리듬탭 권장.

## 6. 준비물 델타 (추가·교체만)

**추가**
- 서약문 대형카드 + 승선 명부 + 스탬프 (0막)
- 삭구 밧줄·지점 라벨 + 코팅 매듭 도해 4장 + 돛천·간이 도르래 (1막)
- 손전등·모스표·배의 종 (2막) — *기존 모스 인프라 재활용*
- 종이 리본 팔찌 4개/팀 + **실물 맹꽁이/다이얼 자물쇠 1개**(게임 유일의 진짜 자물쇠) (3막)
- 반투명 필름 4장/팀 (4막) — *기존 명화필름/repin 자산 재활용, 손전등 2막 겸용*
- LED 티라이트+리모컨·유리병 카트리지·기름통(노란물)·부싯돌·짐 바구니 (5막)

**교체 / 폐기**
- 5개 4자리 숫자 코드 카드 → **전량 폐기**
- "이기면 숫자 도장" 도장 세트 → 실물 아이템(십자가·열쇠·부품) 지급으로 교체
- 3막 하중상자·좌우현 청테이프 선 → **삭제**(병목 압축)

**공통 제작 소품(막 이월로 재사용)**: 나무 십자가 → 1막 지급 → 3막 게이트 → 5막 횃대. 기름·부싯돌 → 3막 지급 → 5막 부품. 심지 → 4막 지급 → 5막 조립.

## 7. 점진 적용 순서 (임팩트 순 3단계)

**1단계 — 숫자 폐기 + 오프라인 3막 (임팩트 최대, 코드 최소)**
- 3·1·5막을 실물·교사 판정형으로 전환: 5개 숫자 자물쇠 제거, 매듭 실물검사(1막)·사슬-해방+실물 열쇠(3막)·부품 조립 점화(5막). **신규 패드 코드 거의 0**(admin 통과·실물 지급만). *유출("2031!" 옆 팀에 외치기)이 즉시 사라진다.*

**2단계 — 목소리·시각 채널 추가**
- 0막 제창+서명, 4막 필름 4겹 합체 도입. 확인버튼 변형(하)·코드필드 문자 허용(하)만 얹으면 됨. 감각 스프레드가 완성된다.

**3단계 — 리듬탭 신규 UI (유일한 개발 부담)**
- 2막 빛신호→리듬탭 위젯 구현(중). 여기까지 와야 6채널이 완전히 다르게 굴러간다. *부담되면 방위 다이얼 리스킨으로 대체 가능.*

## 부록 — 심사 지적 치명 결함, 해소 확인

| 지적 결함 | 해소 |
|---|---|
| 0막·4막 둘 다 사회역학 → 가족 겹침 | 0막을 **목소리**로 밀고 사회역학은 5막 팀간 계승 한 곳에 집중 |
| 신규 패드 UI 남발 | 리듬탭 **1개로 못박음**, 나머지 재활용 |
| 실불 화상·넘어짐 | **LED만**(기름=노란물, 화기 0), 줄다리기·뛰기·인간매듭 잡아당기기 **금지** |
| 3막 교사 병목 | 하중상자 서브과제 삭제, 최정예 배치, 4막 절약 교사 투입 |
| 4막 팀 비동기(교환 상대 없음) | 4막을 **자기검증형 필름 합체**로 채택(팀간 교환은 예비 레버) |
| 연속성 구멍(십자가·부품) | 십자가=1막 지급, 기름·부싯돌=3막, 심지=4막 → 5막 조립 |
| 2막 무한 재시도 | 허용오차 넉넉 + 재발신 3회 캡 + 팀 출발 스태거링 |

**남긴 선택 레버 2개**: ① 2막 리듬탭 ↔ 방위 다이얼(개발 부담 vs 감각 참신도) ② 4막 필름 합체 ↔ 촉각/팀간교환(신뢰성 vs 감각 다양성). 기본값은 각각 리듬탭·필름 합체.
